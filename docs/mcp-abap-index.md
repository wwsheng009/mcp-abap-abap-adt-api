## 💡 关键成功因素

### 1. 始终使用对象锁
```
lock → edit → unlock → activate
```

### 2. 分步骤验证
```
不要一次改太多，每步都验证
```

### 3. 充分利用错误信息
```
检查返回的 messages 数组中的 href 和 shortText
```

### 4. 使用标准 API
```
优先使用 SAP 标准函数和类
```
